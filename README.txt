FormSaveReminder 1.0 by somatonic
A custom module for ProcessWire2.1.


How to install:
- Download the contents of this repository and put the folder renamed as "FormSaveReminder" into your site/modules/ folder
- Login to processwire and got to Modules page and click "Check for new modules". You should see a note that a new module was found. Search for it in the list and click "install".
- Done
